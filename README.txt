Introduction
============

pretaweb.pretamap, is a Popup imagemap for plone content


Installation
============
As with every Plone add-on: add ``pretaweb.pretamap`` to the ``eggs`` and ``zcml``  
option of your buildout configuration and re-run buildout.


Usage
=====
Inside TinyMCE insert a image file then you have to select it image inside the editor window and choose
the imagemap icon from the toolbar of TinyMCE.
